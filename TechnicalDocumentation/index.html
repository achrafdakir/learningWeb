Introduction

Java is a popular, versatile, and object-oriented programming language.

  • Object-Oriented: Java is based on the principles of object-oriented programming (OOP), allowing for modular and reusable code.
  • Platform-Independent: Java code can run on any platform that has the Java Virtual Machine (JVM), making it highly portable.
  • Robust and Secure: Java provides strong memory management, exception handling, and security features.
  • High Performance: Although an interpreted language, Java is optimized through Just-In-Time (JIT) compilation, making it performant.
  • Multi-threaded: Java allows multiple threads to run concurrently, enhancing efficiency in applications.

Known for its portability and performance, Java is widely used in various domains, from web applications to mobile applications, scientific applications, and more. In this documentation, we’ll explore some fundamental concepts of Java.

Printing to the Console

In Java, you can display output to the console using the System.out.println() method. Here’s an example:


                public class HelloWorld {
                    public static void main(String[] args) {
                        System.out.println("Hello, World!");
                    }
                }

This program prints "Hello, World!" to the console.

Conditions

Java supports conditional statements, such as if, else if, and else, to control the flow of the program based on certain conditions. Here’s an example:


                int age = 18;
                if (age >= 18) {
                    System.out.println("You are an adult.");
                } else {
                    System.out.println("You are a minor.");
                }

This code checks the age and prints a message depending on whether the person is an adult or a minor.

Loops

Loops are used to execute a block of code repeatedly. Java provides various loops, such as for, while, and do-while. Here’s an example of a for loop:


                for (int i = 0; i < 5; i++) {
                    System.out.println("Iteration " + i);
                }

This loop prints the current iteration number five times, from 0 to 4.
